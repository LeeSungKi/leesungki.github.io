## 👋 소개

팀 블로그를 직접 운영하기위해 🙌

### 🏃‍♀️ 실행하기

아래 명령어를 실행하여 로컬 환경에 블로그를 실행합니다.

```bash
# Install dependencies
$ npm install

# Start development server
$ npm start
```

<br/>

위 명령어가 문제 없이 실행됐다면 [http://localhost:8000](http://localhost:8000)에서 블로그를 확인

### ❓ 배포하기
```bash
$ npm run deploy

Published라는 메시지를 받으셨다면 배포는 끝
```
